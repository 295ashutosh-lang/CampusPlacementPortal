# Campus Placement Portal

## Project Overview

The Campus Placement Portal is a Full Stack Web Application designed to streamline the campus recruitment process. It provides a centralized platform where students can create profiles, upload resumes, and apply for jobs, while companies can post job openings and manage applicants. The placement administrator can monitor and manage the entire recruitment process through an analytics dashboard.

---

## Problem Statement

Many colleges still rely on spreadsheets, emails, and manual processes to manage campus placements. This often leads to inefficiencies, delayed communication, and difficulty in tracking student applications and company recruitment progress.

The proposed system aims to digitize and simplify the placement process by providing role-based access for students, recruiters, and administrators.

---

## Objectives

- Develop a centralized placement management system.
- Enable secure user authentication using JWT.
- Allow students to browse and apply for jobs.
- Allow recruiters to post and manage job opportunities.
- Provide an admin dashboard for monitoring placement activities.
- Visualize placement statistics through dashboards.

---

## Scope

The system will support three types of users:

- Student
- Company
- Administrator

Each user will have role-specific functionalities.
